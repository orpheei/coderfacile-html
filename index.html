3) Puis, on met le Doctype html

Mais, que veut dire Doctype htlm?
Le doctype html permet en sorte de dire au logiciel que vous utilser le language html. Mais ne nous attardons pas la-dessus, passons à la pratique.

Pour commençer, entrer dans votre 1er ligne de code ‹DOCTYPE html›
Puis, vous allez mettre dans votre 2e ligne de code ‹html› et quelque ligne plus bas, ‹/html›

Oui mais, a quoi cela sert ?
Cela va tout simplment dire ou commence votre code et ou se termine t'il. Tout votre code html se mettra dans cette partie

Ensuite vous alllez metre une balise qui s'appellera ‹head› et qui se terminera par ‹/head› .
On mettre ensuite le code ‹title› "mettez ce que vous voulez ici" ‹title›
puis mettez ‹meta› charset="utf-8"

Mais, a quoi cela sert t'il de mettre cela ?
Tout d'abord le title est l'onglet qui s'affichera tout en bas de la page (mettez ce que vous voulez) et le meta charset= "uft-8" permet de traduire les caractère spéciaux ( #, ç, à, é ...)

Nous allons dans cette partie apprendre à intégrer notre 1er balise .
Tout d'abord, on vat metre la balise ‹body› et quelque ligne plus loin la balise ‹/body› . c'est entre c'est deux partie que nous mettrons tout notre contenu pour notre site , c'est un peu le corps de notre code.

La 2e balise que nous allons voir est la balise ‹h1› qui nous permettra d'intégrer des titre dans notre site web
Elle est composé d'une entrée ‹h1› et d'une fermeture ‹/h1›

Il y a une autre balise qui permettra d'insérer des paragraphe, elle est composée d'une ouverture et d'une fermeture : ‹p› et ‹/p›. c'est entre c'est dans cette balise que nous allons intéger la majorité de notre code
